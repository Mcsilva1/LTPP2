## Sobre o Projeto

Um aplicativo desktop simples para gerenciar alunos e suas matrículas em cursos, desenvolvido com Python, SQLite e Tkinter. O aplicativo possui uma interface gráfica com tema escuro, permitindo adicionar, visualizar e excluir registros de alunos e matrículas.

## Funcionalidades
Gerenciamento de Alunos: Adicione e remova alunos com seus nomes e idades.
Matrículas em Cursos: Matricule alunos em cursos e remova matrículas.
Integração com Banco de Dados: Utiliza SQLite para armazenar dados de alunos e matrículas de forma persistente.
Tema Escuro: Interface moderna com tema escuro para melhor usabilidade e estética.
Tratamento de Erros: Valida entradas e lida com erros do banco de dados de forma amigável.

## Ferramentas Utilizadas

- Python
- SQLite

## Projeto acadêmico

Este projeto é destinado a fins de aprendizado e prática de desenvolvimento web.
